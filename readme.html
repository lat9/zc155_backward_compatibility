What it does

Many of the Zen Cart v1.5.5 admin-level scripts have been modified to use new functions introduced in Zen Cart v1.5.5:

  • zen_build_subdirectories_array
  • zen_sort_array
  • is__writeable
  • set_unwritable
  • updateObjectInfo, present in the objectInfo class

This plugin provides a downward-compatibility path so that any author whose plugin modifies admin-level core files can merge their plugin's changes with the Zen Cart v1.5.5 base files and have a single version of their plugin that properly operates on Zen Cart versions v1.5.0 through v1.5.5!

The compatibility-script is simple in nature; it's auto-loaded just after the admin-level initialization file that defines those functions in Zen Cart v1.5.5. When the script loads, it checks to see if each of the new functions is already defined (i.e. you're running Zen Cart v1.5.5) and, if not, defines that function. That way, the v1.5.5 changes that are merged into a plugin's scripts will operate on pre-Zen Cart v1.5.5 installations.

This plugin also includes the Zen Cart v1.5.5 version of /YOUR_ADMIN/includes/classes/object_info.php, which contains that new class-function.

Note: This plugin is a bit different; it's intended to be included in other plugins' distributions — it provides no functionality in and of itself.

Note, too: For full downward-compatibility, plugin authors should use this plugin in conjunction with ZC1.5.4 - Admin Plugin Backward Compatibility.

Usage

Include the plugin's three files with your plugin's distribution:

  1. /YOUR_ADMIN/includes/auto_loaders/config.zc155_compatability.php
  2. /YOUR_ADMIN/includes/classes/object_info.php (This is the Zen Cart v1.5.5 version of the file)
  3. /YOUR_ADMIN/includes/init_includes/init_zc155_compatibility.php
