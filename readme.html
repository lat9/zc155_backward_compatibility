What it does

Many of the Zen Cart v1.5.5 admin-level scripts have been modified to use a new function named zen_build_subdirectories_array. This plugin provides a compatibility script so that any author whose plugin modifies admin-level core files can merge their plugin's changes with the Zen Cart v1.5.5 version of those files and have a single version of their plugin that properly operates on Zen Cart versions v1.5.0 through v1.5.5!

The script is simple in nature; it's auto-loaded just after the admin-level initialization file that defines the function in Zen Cart v1.5.5. When the script loads, it checks to see if zen_build_subdirectories_array is not already defined (i.e. you're not running Zen Cart v1.5.5) and defines that function. That way, the v1.5.5 changes that are merged into a plugin's scripts will operate on pre-Zen Cart v1.5.5 installations.

Note: This plugin is a bit different; it's intended to be included in other plugins' distributions — it provides no functionality in and of itself.

Note, too: For full downward-compatibility, plugin authors should use this plugin in conjunction with ZC1.5.4 - Admin Plugin Backward Compatibility.

Usage

Include the plugin's two files with your plugin's distribution:

  1. /YOUR_ADMIN/includes/auto_loaders/config.zc155_compatability.php
  2. /YOUR_ADMIN/includes/init_includes/init_zc155_compatibility.php
